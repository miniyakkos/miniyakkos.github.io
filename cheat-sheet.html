GIT Cheat Sheet

by Ayako Kurushima & Brandi Ford

Commands Definitions
git status gives current state of git tracked file, as well as anything that is currently untracked.
git add stages files to be committed. ("git add ." - add everything currently tracked)
git commit saves the changes to all of the added files. "git commit -m "" " is used to give a short message for the committed files. "git commit -v "" " is used to give a verbose message and a sublime document will open.
git push pushes directory to your github. Based on the remotes that have been set up.
git remote -v checks the status of your remotes.
git fetch grabs any changes from the remote directory, based on the remotes that have been set up. example: "git fetch upstream/master"
git merge combines what has been fetched with what already exists in your local directory. example: "git merge upstream/master"
git add -u updates the index - removes as well as modifies index entries to match the working tree. Nifty if you want to delete files.
